# IV.6 Gaming and user experience
Major = 2 points, Minor = 1 point
- Major: Implement a complete web-based game where users can play against each other. (เป็นเกมที่แข่งขันกับฝั่งตรงข้าม)
  - The game can be real-time multiplayer (e.g., Pong, Chess, Tic-Tac-Toe, Card games, etc.). (เกมที่เป็นการแข่งขัน สดๆ แมชต่อแมช เช่น ปอง, หมากรุก, เกมxo, เกมการ์ด และ อื่นๆ)
  - Players must be able to play live matches. (ผู้เล่นสามารถเล่นแมตส์สดๆได้)
  - The game must have clear rules and win/loss conditions. (เกมมีกฎ ชนะ/แพ้ ที่ชัดเจน)
  - The game can be 2D or 3D. (เกมสามารถเป็นได้ทั้ง 2D หรือ 3D เลือกอย่างใดอย่างนึง)
- Major: Remote players - Enable two players on separate computers to play the same game in real-time (เล่นด้วยกันเล่นแบบแยกเครื่องกันได้่)
  -  Handle network latency and disconnections gracefully. (จัดการความหน่วง และ ตัดการเชื่อมต่อ ของผกู้เล่นที่เล่นด้วยกัน ได้ไม่มีปัญหาอะไร)
  -  Provide a smooth user experience for remote gameplay. (ระหว่างที่ผู้เล่น เล่นด้วยกัน ต้องไม่กระตุกเล่นลื่น)
  -  Implement reconnection logic. (ให้มีระบบ เชื่อมต่อใหม่อีกครั้งถ้าเน็ตหลุด หรือเกิดปัญหาบางอย่าง)
- Major: Nultiplayer game (more than two players). (เกมรองรับผู้เล่นมากกว่า 2 คนขึ้นไป)
  - Support for three or more players simultaneously. (รองรับผู้เล่น 3 คน หรือมากกว่านั้น พร้อมกันได้)
  - Fair gameplay mechanics for all participants. (รูปแบบการเล่นของแต่ละคนต้องเท่าเทียมกัน)
  - Proper synchronization across all clients. (ระหว่างโหมดผู้เล่นหลายคนต้องไม่มีปัญหาอะไรสามรารถเชื่อมต่อกันได้ลื่นไหล)
