# Bonus part
The bonus part will be considered only if all required modules have been implemented
corresponding to the minimum of 14 mandatory points.
Each additional module implemented beyond the required 14 points may be considered
as a bonus. (คะแนนที่เกิด ขั้นต่ำ 14 แต้มจะถูกพิจารณาตามข้างล่างนี้)
## For each extra module:
  - It must be fully functional. (ทำงานตาม requirement ของ module ได้ 100%)
  - It must meet the module requirements description. (สิ่งที่ทำใน module นี้ ต้องครงตาม requirement ห้ามออกทะเล)
  - It must add real value to the project. (คะแนนที่ทำมาเพิ่มเติม ต้องสร้าง มูลค่า/ประโยชน์ให้กับโปรเจคด้วย)
  - It must include a proper justification in the README (ต้องบอกเหตุผลที่ชัดเจนของ module เพิ่มเติม ใน README)

Each validated extra module will be taken into account during the review as follows(แต่ละ module คะแนนพิเศษ จะถูกคิดตามนี้):
  - Major modules: 2 points each (Major = 2 คะแนน)
  - Minor modules: 1 point each (Minor = 1 คะแนน)

You can have a maximum of 5 points (e.g., 5 minor modules, or 2 major modules + 1 minor module). (สำหรับคะแนนพิเศษเต็มที่ มีได้ 5 คะแนนเท่านั้น เช่น เรามี 14 คะแนนขั้นต่ำ จะทำเพิ่มได้ 5 คะแนนเท่านั้น 14+5 = 19 คะแนน แล้วแต่ว่าจะแบ่งทำ module แบบไหน เช่น Minor 5 ตัวได้ 5 คะแนน หรือ Major 2 ตัว 4 คะแนน + Minor 1 คัวได้ 5 คะแนนเหมือนกัน)
