# IV.8 Data and Analytics
Major = 2 points, Minor = 1 point
- Major: Advanced analytics dashboard with data visualization. (Dashboard กับ data กราฟ ขั้นสูง)
  - Interactive charts and graphs (line, bar, pie, etc.). (กราฟพวก แผนภูมิรูปทรงต่างๆ เช่น กราฟเส้น, กราฟแท่ง, กราฟวงกลม pie สัดส่วน และอื่นๆ)
  - Real-time data updates. (ข้อมูลที่เข้ามาต้อง Real-time)
  - Export functionality (PDF, CSV, etc.). (มีการสรุปข้อมูลจากกราฟ ที่ export เป็นฟอร์มของ PDF, CSV(excel สกุล CSV) และ อื่นๆเป็นตาราง)
  - Customizable date ranges and filters. (มีให้เราเลือก start date - end date หรือช่วงเวลาเลือกในกราฟได้ และ filters ที่จะดูข้อมูลส่วนที่เราสนใจเช่น ดูช่วงเวลา "10:00" filter เฉพาะผู้หญิงเท่านั้น)
- Minor: Data export and import functionality.
  - Export data in multiple formats (JSON, CSV, XML, etc.). (เอาข้อมูลจากกราฟ export ได้หลายรูปแบบ เช่น JSON, CSV, XML และอื่นๆ)
  - Import data with validation. (นำข้อมูลเข้าระบบ และ ต้องตรวจสอบความถูกต้องของข้อมูล)
  - Bulk operations support. (รองรับการประมวลผลข้อมูลจำนวนมากพร้อมกันได้)
- Minor: GDPR compliance features. (ปฏิบัติข้อมูลของเรา ตามกฎหมายคุ้มครองข้อมูลส่วนบุคคล GDPR)
  - Allow users to request their data. (ให้ผู้ใช้ request ขอข้อมูลของพวกเขาได้)
  - Data deletion with confirmation. (ใช้ระบบ การยืนยันอีกครั้ง เพื่อให้ มั่นใจที่จะลบข้อมูลออกไปได้อย่างไม่มีปัญอะไร)
  - Export user data in a readable format. (Export ข้อมูล user ต้องเป็นรูปแบบที่สามารถเข้าใจได้)
  - Confirmation emails for data operations. (ส่งข้อมูลผ่านอีเมลแจ้งเตือน/ยืนยันการดำเนินการเกี่ยวกับข้อมูลนั้น ไปยัง user เสมอ)
