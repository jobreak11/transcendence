# IV.5 Cybersecurity
Major = 2 points, Minor = 1 point
- Major: Implement WAF/ModSecurity (hardened) + HashiCorp Vault for secrets: (เอาง่ายๆคือ เราเรียกยามสักตนมาดูแลความปลอดภัยให้เว็บของเรา ถ้ามีสิ่งแปลกปลอมมาเจาะ ระบบหลังบ้านเรา จะบล็อก 403 forbidden ทันที)
  - Configure strict Mod Security/WAF. (มีการตั้งค่า ระบบ WAF ได้อย่างแน่นหนา มั่นใจว่ากันได้ แทบจะ 100%)
  - Manage secrets in Vault (API keys, credentials, environment variables), encrypted and isolated. (จัดการข้อมูลลับทั้งหมดผ่าน Vault)
 
# WAF และ HASHICORP Vault คืออะไร
- WAF (Web Application Firewall) เปรียบเสมือน "ยามหน้าประตูเว็บ" ครับ ปกติเวลามี Request วิ่งเข้ามาที่ Web Server ยามตัวนี้จะคอยสแกนดูว่าข้อความที่ส่งมามี "อาวุธ" หรือ "สิ่งแปลกปลอม" ติดมาด้วยไหม เช่น พยายามส่งโค้ดแปลกๆ มาเจาะระบบ (SQL Injection, XSS) ถ้าเจอก็จะบล็อก (HTTP 403 Forbidden) ทันที ก่อนที่ Request นั้นจะไปถึง Back-end ของเรา
- HashiCorp Vault คือ "ตู้เซฟดิจิทัล" ที่เอาไว้เก็บค่าลับเหล่านี้ทั้งหมด โดย Vault จะทำการ เข้ารหัส (Encryption) ข้อมูลลับ และแจกจ่ายให้ Back-end ดึงไปใช้ผ่าน Token / API แบบจำกัดสิทธิ์ ทำให้ในตัวไฟล์โค้ดหรือ Docker Environment ของเราไม่มี Secret หลุดออกมาเลย
